# Instructions

## Requirements
1. Node ^v14.13.0

## Environment
1. `cp frontend/.env.example frontend/.env`
2. Edit `REACT_APP_API_URL` in `frontend/.env` to the API url 

## Installation & Execution
1. `cd backend` && `npm i` && `npm start`
2. `cd frontend` && `yarn` && `yarn start`
